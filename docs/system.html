Messages

The fabric.messages package involves all classes and interfaces relating to messages which can be sent between the different types of fabric nodes. Some of the capabilities provided include:
  • Transaction requests to a store: See PrepareTransactionMessage.java>
  • Ways to update workers and dissemination nodes that an object has been updated: See ObjectUpdateMessage.java>
  • Transaction requests to a store: See PrepareTransactionMessage.java>
  • Worker requests to read an object at a store: See ReadMessage.java>
  • Ways to perform function shipping: See RemoteCallMessage.java>
  • Ability to check whether cached objects are still fresh by requesting response from a store: See StalenessCheckMessage.java>
  • A request from a worker to take ownership of an object from another worker: See TakeOwnershipMessage.java>
